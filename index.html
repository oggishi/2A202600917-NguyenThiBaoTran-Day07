Day 7: Embedding & Vector Store Implementation

📋 Project Overview

This is a Vietnamese Legal Document RAG System that implements semantic search using embeddings and vector storage. The system can retrieve relevant articles from Vietnamese labor and social insurance laws.

✨ Features

  • Document Chunking: Intelligent text splitting with sentence preservation
  • Semantic Embeddings: MockEmbedder (default) or SentenceTransformers
  • Vector Store: In-memory storage with similarity search
  • Metadata Extraction: Automatic Vietnamese legal metadata parsing
  • Benchmark Queries: Pre-configured test queries for validation

📚 Documents

  • README.md — Project documentation
  • data/ — Vietnamese legal documents and design notes
  • docs/ — Evaluation, scoring, and instructor guide

🔧 Technical Stack

  • Language: Python 3.11+
  • Embeddings: MockEmbedder (hash-based) or SentenceTransformers
  • Testing: pytest (42/42 tests passing)
  • Domain: Vietnamese legal documents (Luật BHXH, Luật Lao Động, Luật Việc Làm)
